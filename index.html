The Chain of Truth

Three layers of governance. One single source of truth.

"blue.500": "#1B5BFF"
"spacing.4": "16px"
"radius.md": "12px"
"font.sans": "Inter"
Layer 1

Primitives

The atomic foundation. Raw values live exclusively in primary.json—immutable, platform-agnostic, and the single source of truth for the entire organization.

Layer 2

Semantics

The language of intent. Semantics describe what a value does ("Primary Action"), not what it looks like. This abstraction layer allows for instant, code-free theming.

Text Primary
Surface Brand
Status Success
Layer 3

Components

Blind consumers. Components never hardcode values; they simply request semantic tokens. This strict separation ensures that a button built today will automatically inherit the brand of tomorrow.
